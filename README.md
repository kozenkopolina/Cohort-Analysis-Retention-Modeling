# Cohort Analysis and Retention Modeling

### Мета проєкту:
Дослідження життєвого циклу користувачів та моделювання показників утримання для оцінки лояльності й ефективності каналів залучення.

### Використані інструменти:
* SQL (PostgreSQL) — для очищення та обробки даних.
* Google Sheets — для розрахунків.
* PowerPoint (PDF) — для візуалізації висновків.

### Основні висновки:
- Виявлено суттєвий розрив в утриманні між органічним та промо-трафіком.
- На основі аналізу було обґрунтовано зміну стратегії, що дозволило знизити відтік промо-користувачів.
